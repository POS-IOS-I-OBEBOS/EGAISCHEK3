# DataMatrix Telegram Bot

Приложение с графическим интерфейсом на Python для чтения DataMatrix-кодов из изображений, отправленных в Telegram-бот. Репозиторий включает:

- `bot_app/main.py` — основной скрипт с окном состояния и логами.
- `build_and_run.bat` — сценарий для установки зависимостей и сборки запускаемого файла.
- `requirements.txt` — список зависимостей Python.

## Возможности

- Графический интерфейс с отображением текущего статуса и журнала событий.
- Первый запуск запрашивает и сохраняет токен бота в `config.json`.
- При получении фото или изображения от пользователя бот пытается считать DataMatrix через `pylibdmtx` и возвращает результат.

## Быстрый старт (Windows)

1. Убедитесь, что установлен Python 3.11+ и добавлен в `PATH`.
2. Запустите `build_and_run.bat`. Скрипт создаст виртуальное окружение, установит зависимости и соберёт `dist/datamatrix_bot.exe` с помощью PyInstaller.
3. После завершения сборки запустите `dist/datamatrix_bot.exe`.
4. При первом запуске укажите токен Telegram-бота.

> **Примечание.** В комплект не входит встроенный интерпретатор Python. Если Python не установлен, поместите рядом с `datamatrix_bot.exe` папку `python` с `python.exe`/`pythonw.exe` и стандартной библиотекой или соберите полностью автономную сборку с помощью PyInstaller.

## Сборка собственного `.exe`

Для получения автономного исполняемого файла рекомендуется использовать PyInstaller на машине с доступом в интернет:

```bash
python -m pip install -r requirements.txt pyinstaller
pyinstaller --noconfirm --onefile --windowed bot_app/main.py --name datamatrix_bot
```

Готовый файл появится в `dist/datamatrix_bot.exe`. Чтобы автоматизировать эти шаги, используйте `build_and_run.bat`.

## Разработка

- Запуск из исходников: `python bot_app/main.py`.
- Конфигурация хранится в `config.json`. Для смены токена удалите файл и перезапустите приложение.
